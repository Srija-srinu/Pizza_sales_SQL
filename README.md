# 🍕 Pizza Sales Analysis (SQL Project)

## 📌 Project Overview
This project analyzes pizza sales data to uncover valuable business insights such as sales trends, customer preferences, and product performance.
The dataset was cleaned and prepared using Excel, and SQL queries were used to solve real-world business problems and generate insights.

---

## 🎯 Objectives
- Analyze total orders and revenue
- Identify top-selling pizzas
- Understand customer ordering patterns
- Analyze category-wise performance
- Track sales trends over time

---

## 🗂️ Dataset Description
The dataset consists of the following tables:

- **orders.csv** → Contains order date and time  
- **order_details.csv** → Contains quantity and pizza IDs  
- **pizzas.csv** → Contains price and size details  
- **pizza_types.csv** → Contains pizza names, categories, and ingredients  

---

## 🛠️ Tools & Technologies
- **Excel** → Data cleaning  
- **MySQL** → Data analysis  
- **SQL** → Queries, joins, aggregations  

---

## 📊 Data Preparation (Excel)

Before importing the dataset into MySQL, data cleaning and preprocessing were performed in Microsoft Excel.

### Cleaning Steps:

- Removed duplicate records  
- Handled missing and blank values  
- Standardized column names  
- Corrected data formatting (Date & Numeric fields)  
- Verified data consistency and accuracy  

This ensured reliable and accurate analysis in the SQL phase.

---

## 🔍 Key Analysis Performed

### Basic Analysis
- Total number of orders  
- Total revenue generated  
- Highest priced pizza  
- Most common pizza size  
- Top 5 most ordered pizzas  

### Intermediate Analysis
- Category-wise quantity analysis  
- Orders distribution by hour  
- Category-wise pizza distribution  
- Average pizzas ordered per day  

### Advanced Analysis
- Revenue contribution by category  
- Cumulative revenue over time  
- Top 3 pizzas by revenue (per category)  

---

## 📊 Key Insights
- Total Orders: **21,350**  
- Total Revenue: **817,860+**  
- Large size pizzas are most preferred  
- Classic category generates highest revenue  
- Peak orders occur during evening hours
